# 🔐 Secure Chat App

A simple secure chat application built using Python. It allows users to register, login, and communicate through a basic messaging system with encryption support.

---

## 🚀 Features
- User registration and login system
- Secure message handling with encryption
- Simple chat interface (GUI)
- Local user data storage
- File-based messaging system

---

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- File handling (JSON / TXT)
- Basic encryption logic

---

## 📁 Project Structure
```
secure_chat_app/

│── server.py
│── client.py

│── encryption.py

```

---

## ▶️ How to Run

1. Clone the repository
```bash
git clone https://github.com/vikashdev4/secure_chat_app_by_vikash.git
```

2. Go to project folder
```bash
cd secure_chat_app_by_vikash
```

3. Run server (if required)
```bash
python server.py
```

4. Run client
```bash
python gui_client.py
```
